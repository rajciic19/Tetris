Tetris - Instructions

Welcome to the game of Tetris!

The goal of the game is to stack falling blocks to fill complete rows.

When a row is filled, it disappears and all blocks above it move down.

The game ends when the blocks pile up to the top.

Controlls:

  • Left - Move block to the left
  • Right - Move block to the right
  • Down - Speed up block's descent
  • Up - Rotate block
